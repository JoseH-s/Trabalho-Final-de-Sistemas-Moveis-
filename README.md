## Sobre o projeto

O app permite ao usuário cronometrar suas sessões de trabalho e descanso, salvando o histórico de cada sessão e exibindo estatísticas de desempenho com base na técnica Pomodoro.

---

## Funcionalidades

- Cronômetro com estados de trabalho, pausa e fim de sessão
- Histórico de sessões salvas com detalhes individuais
- Gráfico de pizza com proporção de trabalho e descanso
- Persistência de dados local com SQLite

---

## Tecnologias utilizadas

- **Flutter**
- **Provider** — gerenciamento de estado
- **sqflite** — banco de dados local SQLite
- **fl_chart** — gráfico de pizza
