Pomodoro App

Sobre o projeto
O app permite ao usuário cronometrar suas sessões de trabalho e descanso, salvando o histórico de cada sessão e exibindo estatísticas de desempenho.
Funcionalidades

Cronômetro com estados de trabalho, pausa e fim de sessão
Histórico de sessões salvas com detalhes individuais
Gráfico de pizza com proporção de trabalho e descanso
Persistência de dados local com SQLite

Tecnologias utilizadas

Flutter
Provider (gerenciamento de estado)
SQLite com sqflite (banco de dados local)
fl_chart (gráfico de pizza)

Requisitos atendidos

Navegação entre três telas com passagem de parâmetros
Banco de dados local com SQLite
Componentes com estado (StatefulWidget)
BottomNavigationBar para navegação entre telas
Biblioteca fl_chart do pub.dev
Componentes implementados em arquivos separados com parâmetros
